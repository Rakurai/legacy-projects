# Local Embedding (FastEmbed) Integration Plan

## Context

The MCP doc server currently:
1. **Build time**: loads a pre-computed `embeddings_cache.pkl` (4096-dim, keyed by `member_id`) generated externally by `regen_embeddings.py` calling OpenAI `text-embedding-3-large`
2. **Runtime**: uses an `AsyncOpenAI` client (configured via `EMBEDDING_BASE_URL` / `EMBEDDING_API_KEY` / `EMBEDDING_MODEL` env vars) to embed search queries on the fly for pgvector cosine similarity

The original embeddings were generated by calling `doc.to_doxygen()` on each `Document` in `doc_db.json` — this produces a complete Doxygen-formatted docstring including `@fn`/`@brief`/`@details`/`@param`/`@return`/`@note`/`@rationale` tags.

We want to add FastEmbed (`BAAI/bge-base-en-v1.5`, 768-dim) as a bundled local embedding option that can replace the hosted model end-to-end, controlled by `.env` config.

---

## Design Decisions

| Decision | Choice |
|---|---|
| Model | `BAAI/bge-base-en-v1.5` (768-dim, ~130 MB ONNX) |
| Artifact naming | `embed_cache_{model_slug}_{dim}.pkl` e.g. `embed_cache_bge-base-en-v1.5_768.pkl` |
| Old artifact | `embeddings_cache.pkl` (4096-dim) — **no longer used**, remove from `validate_artifacts` required list |
| Provider selection | `EMBEDDING_PROVIDER` env var: `"local"` \| `"hosted"` \| unset (keyword-only fallback) |
| Build behavior | If matching artifact exists → load it. If missing → generate it using configured provider, save it, then load. |
| DB schema dimension | Dynamic — read from provider config, not hardcoded `Vector(4096)` |
| Async story | Local provider is sync (ONNX CPU); wrap in `asyncio.to_thread()` for server query path. Build script calls synchronously. |

## Embedding Text Construction

The original `regen_embeddings.py` embeds `doc.to_doxygen()` for each entity. This produces text like:

```
/**
 * @fn void do_look(Character *ch, String argument)
 *
 * @brief Handles the 'look' command for examining rooms, objects, and characters.
 *
 * @details Processes look command with multiple sub-modes: look at room, ...
 *
 * @note Uses show_char_to_char for character listings.
 *
 * @param ch The character performing the look
 * @param argument Target to look at (optional)
 * @return void
 */
```

The server's `Document` dataclass (in `artifact_models.py`) has: `brief`, `details`, `params`, `returns`, `notes`, `rationale`, `definition`, `argsstring`, `state`, `usages`, `system`.

The `MergedEntity` has access to both the `DoxygenEntity` (for `name`, `kind`, `signature`) and the `Document`.

We need a `build_embed_text(merged: MergedEntity) -> str` function that reconstructs the same Doxygen-formatted string from these fields, matching the original `to_doxygen()` output shape. This goes in a shared location used by the build script's "generate" path.

### `build_embed_text` logic (port of `Document.to_doxygen()`)

```python
def build_embed_text(merged: MergedEntity) -> str:
    """Build Doxygen-formatted text for embedding, matching original to_doxygen() output."""
    doc = merged.doc
    entity = merged.entity
    lines = ["/**"]

    # Tag line
    tag_map = {
        "function": "@fn", "variable": "@var", "enum": "@enum",
        "class": "@class", "struct": "@struct", "namespace": "@namespace",
        "define": "@def", "group": "@defgroup",
    }
    tag = tag_map.get(entity.kind, "@fn")
    if entity.kind == "function" and doc and doc.definition and doc.argsstring:
        lines.append(f" * {tag} {doc.definition}{doc.argsstring}")
    else:
        lines.append(f" * {tag} {entity.name}")

    if doc:
        if doc.brief:
            lines += [" *", f" * @brief {doc.brief}"]
        if doc.details:
            lines += [" *", f" * @details {doc.details}"]
        if doc.notes:
            lines += [" *", f" * @note {doc.notes}"]
        if doc.rationale:
            lines += [" *", f" * @rationale {doc.rationale}"]
        lines.append(" *")
        if doc.params:
            for pname, desc in doc.params.items():
                lines.append(f" * @param {pname} {desc}")
        if doc.returns:
            lines.append(f" * @return {doc.returns}")

    lines.append(" */")
    return "\n".join(lines)
```

        return self.embedding_provider is not None

        return "embed_cache_none_0.pkl"  # shouldn't be used

### 3. `server/embedding.py` (new)

```python
"""
Embedding Provider abstraction — local (FastEmbed) or hosted (OpenAI-compatible).

One provider is instantiated at startup and shared across all tool calls.
"""
from __future__ import annotations

import asyncio
from typing import Protocol


class EmbeddingProvider(Protocol):
    """Minimal interface for embedding text."""
    @property
    def dimension(self) -> int: ...
    async def embed_query(self, text: str) -> list[float]: ...
    async def embed_documents(self, texts: list[str]) -> list[list[float]]: ...
    # Sync variants for build script
    def embed_query_sync(self, text: str) -> list[float]: ...
    def embed_documents_sync(self, texts: list[str]) -> list[list[float]]: ...


class LocalEmbeddingProvider:
    """FastEmbed ONNX-based local embeddings."""

    def __init__(self, model_name: str = "BAAI/bge-base-en-v1.5") -> None:
        from fastembed import TextEmbedding
        self._model = TextEmbedding(model_name=model_name)
        # Probe dimension
        probe = list(self._model.embed(["test"]))[0]
        self._dimension = len(probe)

    @property
    def dimension(self) -> int:
        return self._dimension

    def embed_query_sync(self, text: str) -> list[float]:
        return list(self._model.embed([text]))[0].tolist()

    def embed_documents_sync(self, texts: list[str]) -> list[list[float]]:
        return [vec.tolist() for vec in self._model.embed(texts, batch_size=32)]

    async def embed_query(self, text: str) -> list[float]:
        return await asyncio.to_thread(self.embed_query_sync, text)

    async def embed_documents(self, texts: list[str]) -> list[list[float]]:
        return await asyncio.to_thread(self.embed_documents_sync, texts)


class HostedEmbeddingProvider:
    """OpenAI-compatible hosted embedding API."""

    def __init__(self, base_url: str, api_key: str, model: str, dimension: int) -> None:
        from openai import AsyncOpenAI, OpenAI
        self._async_client = AsyncOpenAI(base_url=base_url, api_key=api_key)
        self._sync_client = OpenAI(base_url=base_url, api_key=api_key)
        self._model = model
        self._dimension = dimension

    @property
    def dimension(self) -> int:
        return self._dimension

    def embed_query_sync(self, text: str) -> list[float]:
        response = self._sync_client.embeddings.create(
            model=self._model, input=text, encoding_format="float"
        )
        return response.data[0].embedding

    def embed_documents_sync(self, texts: list[str]) -> list[list[float]]:
        response = self._sync_client.embeddings.create(
            model=self._model, input=texts, encoding_format="float"
        )
        return [d.embedding for d in response.data]

    async def embed_query(self, text: str) -> list[float]:
        response = await self._async_client.embeddings.create(
            model=self._model, input=text, encoding_format="float"
        )
        return response.data[0].embedding

    async def embed_documents(self, texts: list[str]) -> list[list[float]]:
        response = await self._async_client.embeddings.create(
            model=self._model, input=texts, encoding_format="float"
        )
        return [d.embedding for d in response.data]


def create_provider(config) -> EmbeddingProvider | None:
    """Factory: instantiate the configured provider, or None."""
    if config.embedding_provider == "local":
        return LocalEmbeddingProvider(model_name=config.embedding_local_model)
    elif config.embedding_provider == "hosted":
        if not config.embedding_base_url:
            return None
        return HostedEmbeddingProvider(
            base_url=config.embedding_base_url,
            api_key=config.embedding_api_key or "default",
            model=config.embedding_model or "text-embedding-3-large",
            dimension=config.embedding_dimension,
        )
    return None
```

### 9. `build_helpers/embeddings_loader.py` — rewrite

```python
def get_embed_cache_path(artifacts_dir: Path, config: ServerConfig) -> Path:
    """Return path for the config-specific embed cache artifact."""
    return artifacts_dir / config.embed_cache_filename


def load_embeddings(artifacts_dir: Path, config: ServerConfig) -> dict[str, list[float]] | None:
    """Load embeddings from model-specific cache. Returns None if file missing."""
    cache_path = get_embed_cache_path(artifacts_dir, config)
    if not cache_path.exists():
        log.info("Embed cache not found, will generate", path=str(cache_path))
        return None
    log.info("Loading embed cache", path=str(cache_path))
    with cache_path.open("rb") as f:
        return pickle.load(f)


def generate_embeddings(
    merged_entities: list[MergedEntity],
    provider: EmbeddingProvider,
    artifacts_dir: Path,
    config: ServerConfig,
) -> dict[str, list[float]]:
    """
    Generate embeddings for all entities using the configured provider.
    Saves to model-specific cache file. Returns the embeddings dict.
    """
    from build_helpers.embed_text import build_embed_text

    texts, ids = [], []
    for merged in merged_entities:
        text = build_embed_text(merged)
        if text:
            texts.append(text)
            ids.append(merged.entity_id)

    log.info("Generating embeddings", count=len(texts), model=config.embedding_local_model)
    vectors = provider.embed_documents_sync(texts)
    embeddings = dict(zip(ids, [v for v in vectors]))

    cache_path = get_embed_cache_path(artifacts_dir, config)
    with cache_path.open("wb") as f:
        pickle.dump(embeddings, f)
    log.info("Embed cache saved", path=str(cache_path), dimension=provider.dimension)

    return embeddings


def attach_embeddings(merged_entities, embeddings):
    # Unchanged — same as current
```

## Risk Notes

- **First-run cold start**: FastEmbed downloads the ONNX model on first use (~130 MB). Subsequent runs use the local cache (`~/.cache/fastembed/`). Log clearly when this happens.
- **Build time**: ~5300 entities × 768-dim local inference ≈ 1-3 minutes on CPU. Acceptable for offline build. Consider adding a progress bar (tqdm or manual logging every N batches).
- **Memory**: 5300 × 768 × 4 bytes ≈ 16 MB in-memory. Negligible.
- **Query latency**: Single-text local embed ≈ 5-20ms on modern CPU. Fine for interactive use under `asyncio.to_thread()`.
- **Dimension mismatch guard**: At server startup, if `embedding_provider.dimension != config.embedding_dimension`, fail fast with a clear error message.
